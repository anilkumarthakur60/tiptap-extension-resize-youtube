# Installation

## Using npm

```bash
npm install @anilkumarthakur/tiptap-extension-resize-youtube
```

## Using yarn

```bash
yarn add @anilkumarthakur/tiptap-extension-resize-youtube
```

## Using pnpm

```bash
pnpm add @anilkumarthakur/tiptap-extension-resize-youtube
```

## Requirements

This extension requires the following peer dependencies:

- `@tiptap/extension-youtube` (^3.15.3)
- `@tiptap/pm` (^3.15.3)

Make sure you have these packages installed in your project:

```bash
npm install @tiptap/extension-youtube @tiptap/pm
```
